# Moneycontrol Automation Framework

## Overview
This project is a Selenium + Pytest automation framework built to automate UI flows on https://www.moneycontrol.com.

It follows Page Object Model (POM) design and supports:
- UI automation
- Data-driven testing
- Cross-browser execution
- Screenshot capture on failures
- Pytest and Allure reporting

## Tech Stack
- Python 3.12
- Selenium WebDriver
- Pytest
- WebDriver Manager
- Allure Reports
